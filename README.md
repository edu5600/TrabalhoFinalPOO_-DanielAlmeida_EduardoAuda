# TrabalhoFinalPOO_-DanielAlmeida_EduardoAuda

**1. Descrição do jogo:**

O jogo se trata de um personagem que deve escalar uma masmorra de **n (10–30) andares** até sua morte ou vitória. Em cada andar, podem ser escolhidas duas opções: **encontro hostil (batalha)** e **enfermaria (cura)**. O nível de dificuldade dos combates aumenta progressivamente conforme o avanço pelos andares.

**2. Como executar:**

Baixe o arquivo em sua máquina e execute o arquivo **Main**.

**3. Conceitos aplicados:**

Utilizamos herança, classes abstratas, interfaces, tratamento de exceções, composição, matrizes e listas (fila).

**4. Detalhamento do item criativo:**

A inspiração inicial foi o jogo web **Pokelike**.
